# Advent of Code 2023

## Quick setup
``` shell
rustup default stable
cargo install cargo-generate
brew install just
```

## Some commands
Create package for a new day:
```shell
just create <day>
```

Execute binary for a given day and task:
```shell
just run <day> <part>
```
where `part` is either `part01` or `part02`
